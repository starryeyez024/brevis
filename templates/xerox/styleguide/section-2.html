# 2.5 - Primary Buttons

Most button use cases should use one of the modifier classes below. Both 'button-primary' and 'button-primar--modifier' doesn't need to be added to the markup. Adding only the modifier class works fine.

Available modifiers 
.button-primary--one

Button Primary: color one

 
.button-primary--two

Button Primary: color two

 
.button-primary--three

Button Primary: color three

 
.button-primary--four

Button Primary: color four

 
.button-secondary

Button Secondary

 
.button--large

Larger button alternative

# 2.7 - Classy Panel Styles: Text Styles

Available modifiers 
.text-white

Turn all text white

 
.text-navy

Turn all text Butler blue
Default

First-Level Header; Any header elements may include links.

The main header above is an h1 element.

Second-Level Header; Any header elements may include links.

The secondary header above is an h2 element, which may be used for any form of important page-level header. More than one may be used per page. Consider using an h2 unless you need a header level of less importance, or as a sub-header to an existing h2 element.
